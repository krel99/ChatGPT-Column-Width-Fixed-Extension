Privacy Policy

Extension: ChatBots Two Columns UI Fix

Effective date: 2026-02-05

This policy explains what information the extension can access and how it is handled.

What the extension does

This extension adjusts the layout of supported chatbot websites on large screens by injecting CSS and a small amount of page logic (for example: applying a two-column conversation layout, adding spacing controls, and optionally hiding or collapsing certain UI sections such as “Related” panels).

Where it runs

The extension runs only on the specific websites listed in its matches configuration (for example: chatgpt.com, chat.openai.com, claude.ai, gemini.google.com, perplexity.ai, grok.com / x.com grok pages, and other listed chatbot domains). It does not run on arbitrary websites.

Permissions

The extension requests the storage permission to save your layout preferences. It does not request broad host permissions beyond the supported site list.

Information the extension may access

To apply layout changes, the extension may read and modify webpage elements on supported sites (the page DOM), for example to classify chat messages for layout or to attach UI toggles. This processing occurs locally in your browser.

Data collection

The extension does not collect, sell, or transfer personal information.

Data transmission

The extension does not send data to the developer or to third parties. It does not include analytics, advertising, tracking, telemetry, or remote logging, and it does not make network requests for data collection.

Data storage

The extension stores only user preferences required for functionality (such as enabling/disabling the side-by-side layout and the selected column ratios / spacing). These preferences are saved using Chrome’s extension storage (for example chrome.storage.sync), so they may sync across your Chrome browsers if you have sync enabled.

The extension does not store your chat content, browsing history, or full page content.

Third-party services

The extension does not use third-party analytics, advertising, or tracking services.

Children’s privacy

The extension is not directed to children under 13 and does not knowingly collect personal information from children.

Changes to this policy

If this policy changes, the updated version will be posted at this same URL with a new effective date.
